Formspree Configuration

To use the email system, you need to configure Formspree:

1. Go to https://formspree.io and create an account 2. Create a new form in your dashboard 3. Copy your form endpoint (looks like: https://formspree.io/f/xvgpkqyw) 4. Update success.html line ~400 and replace 'YOUR_FORMSPREE_ENDPOINT' with your form ID 5. Test the email system using the buttons above
